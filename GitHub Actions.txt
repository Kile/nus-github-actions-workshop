GitHub Actions

- need a .github/workflows/*.yml file
- always need an "on"
	- can specify which branches the jobs run on, and on which actions
- "jobs":
	- "runs-on": specifies which platform to run the job on (recommend "ubuntu-latest")
	- "steps": specifies the actions from the marketplace that you want to run
- can use nektos/act to run jobs locally
